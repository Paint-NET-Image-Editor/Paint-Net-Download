## Installation & Setup Guide for Paint.NET

#### System Compatibility

- **Supported Windows versions:** Windows 10, 11 (64-bit required)
- **Architecture support:** 64-bit (x64) systems only
- **.NET Framework:** Requires .NET Desktop Runtime 6.0 or later (automatically installed if needed)
- **Disk space required:** Approximately 100-150 MB for the application

#### Installation Methods

**Direct Installation:**

1. Download the Paint.NET installer package for Windows.
2. Run the downloaded `.exe` installer file.
3. Follow the on-screen setup wizard to complete the installation.
4. Launch Paint.NET from the Start Menu or desktop shortcut.

**Portable Version:**

1. Download the portable version package (ZIP archive) if available.
2. Extract the archive contents to a folder or USB drive.
3. Run the `PaintDotNet.exe` file directly; no installation is required.

#### First-Time Setup & Configuration

1. **Launch Application:** On first launch, Paint.NET presents a clean interface with tools panel, history window, and layers panel.
2. **Default Settings:** Configure default image settings, file associations, and autosave preferences in the Settings dialog (File → Settings).
3. **Plugin Support:** Optional plugins can be installed by copying files to the `Effects` or `FileTypes` folders in the program directory.

#### Core Image Editing Operations

1. **Opening Images:**
   - Use File → Open to load existing images for editing.
   - Drag and drop images directly into the application window.
   - Supports common formats including PNG, JPEG, BMP, GIF, TIFF, and PSD (with plugin).

2. **Layer Management:**
   - This layer-based editor displays each layer in the Layers window.
   - Add, delete, duplicate, merge, and reorder layers as needed.
   - Adjust layer opacity and blending modes for compositing effects.

3. **Basic Adjustments:**
   - Access brightness, contrast, hue, and saturation adjustments via the Adjustments menu.
   - Levels, curves, and black/white adjustment tools for advanced tonal control.
   - Photo retouching tool capabilities include red-eye removal and color correction.

#### Key Features & Advanced Options

- **Selection Tools:** Rectangle, ellipse, lasso, magic wand, and color range tools for precise image manipulation.
- **Drawing Tools:** Brush, pencil, airbrush, clone stamp, and gradient tools for digital art software functionality.
- **Effects Library:** Extensive collection of filters including blur, sharpen, distortion, noise, and artistic effects for photo manipulation tool capabilities.
- **Plugin Ecosystem:** Third-party plugins expand functionality with additional effects, file format support, and automation features.
- **Unlimited Undo:** History window tracks all changes for non-destructive editing.

#### Troubleshooting

- **Application Won't Start:** Verify .NET Desktop Runtime is installed. Reinstall Paint.NET to fix corrupted installation.
- **Slow Performance with Large Images:** Increase memory allocation in Settings → Advanced. Consider using 64-bit version for better performance.
- **Plugins Not Working:** Ensure plugins are compatible with your Paint.NET version. Place plugin files in correct folders (Effects or FileTypes).
- **File Association Issues:** Set Paint.NET as default image editor in Windows Settings or within the application's Settings dialog.

#### Updates & Maintenance

- Paint.NET notifies users when updates are available via the built-in updater.
- Manual updates require downloading and running the latest installer.
- Plugin updates require checking individual plugin sources.

#### Support Resources

- Built-in help documentation accessible via F1 or Help menu.
- Official website provides tutorials, forums, and plugin resources.
- Active user community for troubleshooting and advanced techniques.

## About Paint.NET

Paint.NET is a versatile image editor for Windows that began as an undergraduate project and has grown into a full-featured raster graphics editor trusted by millions. This image processing software combines the simplicity of basic editing tools with the power of a layer-based editor, making it suitable for both casual users and serious digital artists. As a comprehensive photo editing software, it offers a wide range of adjustments, effects, and selection tools that compete with more expensive applications. The digital art software capabilities are enhanced by support for pressure-sensitive tablets and a rich ecosystem of community-developed plugins. For users needing a graphic design software solution without the complexity of professional suites, Paint.NET provides a balanced image editing application that handles everything from simple photo retouching to complex image composition with intuitive controls and reliable performance.
